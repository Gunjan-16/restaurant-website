# 🍽️ Restaurant Website - Roti & Royalty

This is a royal-themed Indian restaurant website created using **HTML** and **CSS**.

## 🌟 Features

- Royal hero banner with tagline
- Multiple food menu sections (Thalis, Street Food, Tandoori, etc.)
- Image transitions and hover effects
- Contact & Social Media links
- Mobile-friendly and responsive

## 🛠️ Built With

- HTML5
- CSS3
- No JavaScript used

## 🖥️ How to View

1. Click the green **Code** button → **Download ZIP**
2. Extract the ZIP file
3. Open `index.html` in your browser
